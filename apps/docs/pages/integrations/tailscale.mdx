# Tailscale

## Scanner

The Tailscale scanner syncs devices from your Tailscale network into Ctrlplane,
allowing you to deployment upgrade and manage devices in your Tailscale network.

### Usage

To sync in all Tailscale devices, run:

```bash
ctrlc sync tailscale \
    --tailscale-key <TAILSCALE_KEY> \
    --tailnet <TAILSCALE_NET> \
    --interval 5m
```
